# Testable Profile Card

This ias an accessible, responsive Profile Card component/page using semantic HTML and modern frontend patterns (HTML/CSS/vanilla JS only).

## Getting Started

Follow these steps to run the project locally:

###  Clone the repository
```bash
git clone https://github.com/Isaacluka/Testable-Profile-Card.git
---

#### Navigate into the Project Folder  

cd Testable-Profile-Card
```md

### Optional: Use a Live Server

For a better development experience, use a live server:

#### Using VS Code:
1. Install the **Live Server** extension
2. Right-click on `index.html`
3. Click **"Open with Live Server"**
